# 📂 FASE BONUS: WAWASAN NOC ENGINEER MODERN (OPSIONAL)

## 🛠️ SKILL-26: Pengenalan Otomatisasi Jaringan Dasar (Network Automation)
*   **Kategori:** Otomatisasi Efisiensi Kerja & Skrip Manajemen Massal
*   **Alat Kerja (Tools):** Bahasa Pemrograman Python (Library Netmiko), Terminal Linux.
*   **Fokus Materi:** Memahami bahwa NOC modern tidak mengecek 100 router satu per satu secara manual. Siswa diperkenalkan dasar skrip Python sederhana untuk masuk ke banyak router secara otomatis guna menarik data log atau mengubah konfigurasi massal dalam sekali klik.

## 🛠️ SKILL-27: Dasar Pemantauan Infrastruktur Cloud (AWS / GCP / Azure)
*   **Kategori:** Arsitektur Cloud Virtualization & Cloud Metrics Monitoring
*   **Alat Kerja (Tools):** AWS CloudWatch / Google Cloud Monitoring Dashboard, Docker.
*   **Fokus Materi:** Memahami bahwa infrastruktur saat ini tidak hanya berbentuk server fisik di dalam kantor, melainkan berada di komputasi awan (Cloud). Siswa belajar membaca metrik kesehatan server virtual dan memantau status aplikasi berbasis kontainer (Docker).

## 🛠️ SKILL-28: Deteksi Serangan Siber Dasar (DDoS & Brute Force Alerts)
*   **Kategori:** Keamanan Jaringan Dasar & Kolaborasi Tim SOC (Security Operations Center)
*   **Alat Kerja (Tools):** Grafana Security Alerts, Wireshark, Firewall Logs.
*   **Fokus Materi:** Melatih ketajaman analisis untuk membedakan kasus internet lambat akibat kabel rusak dengan internet lambat akibat serangan banjir paket data siber (**DDoS Attack** atau **Brute Force**). Siswa diajari alur eskalasi darurat menuju ke tim SOC (*Security Operations Center*) saat terdeteksi indikasi serangan berbahaya.
